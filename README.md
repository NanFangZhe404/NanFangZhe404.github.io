## 启动流程

```shell
npx degit clark-cui/vitepress-blog-zaun my-vitepress-blog
cd my-vitepress-blog
pnpm i # If you didn't install pnpm,you can run: npm install -g pnpm
pnpm dev 
```

If you want to view rss result, you should run

```shell
pnpm run build
pnpm run serve
```
